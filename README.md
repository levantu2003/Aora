# How to use
## Directly
Use your android phone and open this link to install the apk:

https://expo.dev/accounts/kaiservjppro/projects/Aura/builds/e2c3e366-ac13-4920-8e8c-d06a94785c03

## Developing

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```
